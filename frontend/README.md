# PizzUM & BurgUM - Frontend

Sistema web para restaurante online de pizzas y hamburguesas personalizadas.

## Tecnologías

- **Next.js 15.5.4** (App Router)
- **React 19** + TypeScript
- **Tailwind CSS 3**
- **Axios** - Peticiones HTTP
- **React Query** - Estado del servidor
- **Phosphor Icons** - Iconografía
- **Tremor** - Gráficos (admin)

## Requisitos Previos

- Node.js 18 o superior
- npm

## Instalación
```bash
# Clonar el repositorio
git clone <url-del-repo>

# Entrar a la carpeta frontend
cd frontend

# Instalar dependencias
npm install
